# Flet_ERP

El proposito principal de este repositorio es testear el framework Flet en aplicaciones de escritorio con Python, utilizando como idea un software ERP generalista.
El software debe incluir lo siguiente:
- [ ] Inventario
- [ ] Ventas
- [ ] Clientes
- [ ] Proveedores
- [ ] Gestion financiera
- [ ] Punto de venta
- [ ] Recursos humanos
- [ ] Analytics
